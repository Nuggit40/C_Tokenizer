# C_Tokenizer
## Program Information
The program accepts user input strings of characters, numberes, punctuation, and some functions and returns token types. 
## Token types
- *Alphabetic Characters and Word Strings*
- *Numbers*
- *Hex Numbers*
- *Operators*
